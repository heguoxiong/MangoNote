# Pytorch

## 常用模块

```shell
torch.utils.data：提供了有关数据处理的工具

torch.nn：定义了大量神经网络的层
torch.nn.init：定义了各种初始化方法

torch.optim：优化器

torchvision：服务于PyTorch深度学习框架，主要用于构建计算机视觉模型
torchvision.datasets：一些加载数据的函数及常用的数据集接口
torchvision.models：包含常用的模型结构（含预训练模型），例如，AlexNet，VGG，ResNet等
torchvision.transforms：常用的图片变换，例如裁剪，旋转等
torchvision.utils：其它的一些有用的方法
```
